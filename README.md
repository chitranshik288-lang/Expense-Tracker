# 💰 Expense Tracker

A simple and responsive **Expense Tracker Web Application** built using HTML, CSS, and JavaScript that helps users manage their income and expenses efficiently.

## 🚀 Features

- Add Expenses
- Edit Expenses
- Delete Expenses
- Search Expenses
- Filter by Category
- Local Storage Support
- Date Tracking
- Clear All Expenses
- Responsive UI

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Local Storage API
